# Warhammer 40K —《為了帝皇》

橫向卷軸動作遊戲，純前端單檔（`index.html`），手機優先 + PWA 可離線。

## ▶️ 立即遊玩

**https://rai0603.github.io/warhammer40k/**

手機可「加入主畫面」當 App 全螢幕橫向玩。

## 玩法

- 太空陸戰隊橫向闖關，5 關 + 5 大魔王
- 遠程 4 種（爆彈槍 / 火焰槍 / 電磁砲 / 槍榴彈）+ 近戰 3 種（動力劍 / 動力錘 / 動力拳），可切換、可升級到 LV5
- 撿拾掉落物補血、加能量、升級武器，集滿能量自動強化
- 稀有「★ 帝皇恩賜」短時間無敵 + 雙倍火力

## 本地開發

```bash
python3 -m http.server 8000
# 開 http://localhost:8000/
```
